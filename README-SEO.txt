# SEO y Estructura Técnica de Top Tier

Este repositorio contiene los archivos clave para el posicionamiento orgánico de https://www.toptier.com.ar

## Archivos incluidos

- index_TOP_TIER_FINAL_FULL.html → Versión optimizada del sitio (SEO + contenido invisible)
- favicon.ico → Favicon multipropósito y optimizado (ubicado en la raíz del dominio)
- sitemap_TOP_TIER.xml → Sitemap con imágenes embebidas para mejorar el tráfico de Google Imágenes
- robots.txt → Guía de rastreo para bots: permite SEO y bloquea scrappers y bots abusivos

## Recomendaciones

1. Subir el favicon a la raíz y asegurarse de que esté enlazado en el <head> del HTML.
2. Mantener actualizado el sitemap si se agregan nuevas imágenes importantes.
3. En Search Console, subir sitemap.xml y solicitar indexación del sitio tras cada cambio.
4. En el HTML, los textos invisibles (`<div style="display:none;">`) refuerzan el SEO sin afectar el diseño visual.
5. No eliminar los alt de imágenes, ayudan al posicionamiento por búsquedas visuales.
